Mobile Connect Demo
-------------------

This is an example application to demonstrate the use of the mobile connect sdk.

It is an Apache Maven based project and requires Apache Maven (https://maven.apache.org) to be installed.

It uses Spring Boot to host the demo application, Spring Boot is not required to use mobile-connect-sdk.

Running
-------

The mobile-connect-sdk application needs to be installed in the local Apache Maven repository.
(HINT: run 'mvn install' in the root directory of the mobile-connect-sdk project)

In the root of the demo project run the following:

mvn spring-boot:run

and then navigate to http://localhost:8080/ in your browser.
